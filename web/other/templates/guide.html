Why choose password vault?

PasswordVault is a password manager, to put it simply. Apart from password storage, it can also be used for a variety of other tasks, such as storing notes and contacts (securely, of course), sharing passwords with contacts, and, finally, generating super secure passwords. Most password managers that perform this function charge a fee, but that is not the case with PasswordVault, which is always free. Now that we've covered what it is and what it can do, it's time to get down to business.

Quick start guide

It's a five-step process that's easy to follow.

Step 1: Install django and other dependencies.

First step is to install django in your system, as PasswordVault uses django for backend. It is a simple pip install to do, if you have python3 in your system. Once that is done then 2 other modules must be installed in the system: cryptography and uuid. the command is as follows: pip install cryptography pip install uuid

Step 2: Download he application from github.

Simply go to this url and git clone the repository. Once that is done, open the repository in the terminal and navigate to the "web" folder.

Step 3: Run the server.

Once you are in this folder and every required module is installed, you can run server to start the app with the following command: python manage.py runserver

Step 4: Register an account if you are a new user or login if you alredy have an account.

Go to accounts page and click on register to register a new account on the app. Once you are inside the app, make necessary changes within the app and to save those changes make sure to download your vault (Download userfile).

Step 5: It's pretty self-explanatory what to do once you're inside the app.

Yes, its that simple.
